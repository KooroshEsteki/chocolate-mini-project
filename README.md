# Chocolate Mini Project

This project analyzes the Chocolate Bar Ratings dataset using Python.

## Project Tasks

This mini project includes:

- Reading the dataset with Pandas
- Checking for missing values
- Removing missing value entries
- Counting the number of tuples
- Counting unique company names
- Counting reviews made in 2013
- Checking missing values in the Bean Type column
- Creating a histogram of the Rating column
- Converting Cocoa Percent from string to numerical values
- Plotting Cocoa Percent against Rating
- Normalizing the Rating column
- Listing companies ordered by average rating
- Encoding company names and company locations

## Dataset

Dataset source:

https://www.kaggle.com/datasets/rtatman/chocolate-bar-ratings
